# NEOGEN Payback Calculator

Статический сайт-калькулятор окупаемости для GitHub Pages.

## Структура

```text
neogen-site/
├─ index.html
├─ assets/
│  ├─ css/
│  │  └─ style.css
│  └─ js/
│     ├─ data.js
│     ├─ utils.js
│     └─ calculator.js
└─ README.md
```

## Как загрузить на GitHub

1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы из этой папки в корень репозитория.
3. Откройте `Settings` → `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Выберите ветку `main` и папку `/(root)`.
6. Нажмите `Save`.
7. Через короткое время сайт будет доступен по ссылке GitHub Pages.

## Как обновлять

- заменяйте `index.html`, `style.css` и JS-файлы;
- делайте commit изменений;
- GitHub Pages автоматически обновит сайт.
